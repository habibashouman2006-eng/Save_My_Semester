# 💜 Save My Semester
**Plan · Prioritize · Succeed**

## Project Idea
An academic decision support system that helps students prioritize tasks,
recover from overload, and manage deadlines realistically.

## Features
- Add assignments with deadline, difficulty, grade weight, and progress
- Automatic priority scoring algorithm
- Ranked task calendar view
- Overload detection (daily & weekly)
- What-If Simulator (skip task / limited hours / finish first)
- Task Clock + Pomodoro Break Timer
- Procrastination Pattern Tracker
- Small Wins Mode for when you feel overwhelmed
- Auto-reschedule missed/overdue tasks
- Burnout & Stress Estimator

## How to Run

### Requirements
- A C++ compiler (g++ recommended)
- C++17 or later

### Compile
```bash
g++ -std=c++17 -o save_my_semester save_my_semester.cpp
```

### Run
```bash
./save_my_semester
```
On Windows:
```bash
save_my_semester.exe
```
